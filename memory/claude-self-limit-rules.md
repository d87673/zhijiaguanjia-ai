# Claude 上下文强制约束规则

## 1. 接续上下文原则
所有任务必须承接当前会话历史上下文，默认接续上一条指令的工作进度，禁止重新全局扫描项目。

## 2. 禁止自动遍历
严禁自动遍历项目全部文件夹，仅在用户明确指定路径时，才读取对应文件。

## 3. 对话历史限缩
对话历史仅保留最近 5 轮记录，更早内容压缩为极简摘要，不再带入请求。

## 4. 全局扫描许可
只有用户手动下达「全项目整体排查」指令，才允许扫描全部目录。
